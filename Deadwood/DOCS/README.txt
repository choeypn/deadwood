To compile our code for Deadwood, run the following command in the sources 
directory with all the .java class files 

javac *.java 

To run the program in the linux shell, navigate to the parent directory 
and run the program with the following command

java Deadwood.Deadwood 2

Where the 2 is the input of players (only 2-3 valid for this version)

Or alternatively, import the project folder into an IDE (such as IntelliJ
or eclipse) and run from there. 

The rules of the game are self evident from print statements (what you can and
cannot do)
